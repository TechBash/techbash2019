# Be a Hero On Day 1 With ASP.Net Boilerplate

Start your next ASP.Net project with a fully functional, attractive site with scores of best practices implemented, whether you choose React, Angular or Vue as your front-end or ASP.Net Core or even ASP.NET MVC 5.x as your back end. In this session you'll learn everything you get out of the box with ASP.Net Boilerplate, a free, open source web application framework. Don't miss this opportunity to be a hero on day 1 of your next web project.
